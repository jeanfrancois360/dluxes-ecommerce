# NextPik Project Context

## Quick Facts
- **Project:** NextPik
- **Version:** 2.3.0
- **Deadline:** January 3, 2026

## Tech Stack
- Frontend: Next.js 15, React 19, Tailwind
- Backend: NestJS 10, Prisma 6.1, PostgreSQL 16
- Infrastructure: Docker, Redis, Meilisearch, Supabase

## Ports

- Redis: 6379
- Meilisearch: 7700

## URLs

- Prisma Studio: http://localhost:5555

## Commands
pnpm dev:web    - Frontend only
pnpm dev:api    - Backend only
